# Web Scraping Dashboard

A Flask-based web application that scrapes website data, stores it in a MySQL database, and displays the results in an interactive dashboard.

## Features

* User Authentication
* Web Scraping
* MySQL Database
* Search & Filter
* Responsive Dashboard

## Technologies Used

* Python
* Flask
* MySQL
* HTML, CSS, JavaScript
* Bootstrap 5

## Run the Project

```bash
pip install -r requirements.txt
python app.py
```
